# atividade-git

## Descrição
Repositório criado para a atividade de controle de versão com Git e GitHub.

## Objetivos
- Aprender os conceitos básicos do Git e GitHub
- Criar e gerenciar repositórios
- Praticar commits, branches, pull requests e merges

## Tecnologias
- Git
- GitHub

## Como usar
1. Clone o repositório: `git clone https://github.com/SabrinafBernardi/atividade-git.git`
2. Entre na pasta: `cd atividade-git`
